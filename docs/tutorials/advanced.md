## 环境准备 ##
* 下载RT-Thread源码

    RT-Thread 代码托管在github上，通过以下git命令下载RT-Thread源码到本地。

```
    git clone https://github.com/RT-Thread/rt-thread
```

* 下载RT-Thread ENV工具


    [https://github.com/RT-Thread/rt-thread](http://#)
